# Clawdbot on Railway

Deploy Clawdbot AI assistant to Railway with Gemini support.

## Quick Deploy

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/template/clawdbot)

## Manual Setup

1. Fork/Clone this repo
2. Create new Railway project from GitHub
3. Set environment variables (see below)
4. Deploy!

## Environment Variables

Required:
- `GOOGLE_API_KEY` - Your Google Gemini API key

Optional:
- `PORT` - Default: 18789
- `NODE_ENV` - Default: production

## After Deployment

1. Railway will generate a public URL (e.g., `your-app.up.railway.app`)
2. Access Clawdbot dashboard at: `https://your-app.up.railway.app`
3. Configure agents and channels via the web UI
